**Portfólio Profissional** 
Este repositório contém o meu portfólio pessoal, desenvolvido para apresentar minha trajetória como *Desenvolvedora Web*, meus projetos práticos e minhas habilidades técnicas.

Tecnologias Utilizadas
Para a construção deste portfólio, utilizei:

HTML5 Semântico: Para garantir acessibilidade e SEO.

CSS3 Moderno: Com uso de Variáveis (CSS Variables), Flexbox, CSS Grid e animações @keyframes.

JavaScript (ES6+): Manipulação dinâmica do DOM, integração com API do WhatsApp e eventos interativos.

Design Responsivo: Implementação de Media Queries para adaptação total em dispositivos móveis.

Projetos em Destaque
O portfólio apresenta três projetos principais que demonstram minhas competências:

Jogo - Ponte de Vidro: Lógica de programação pura e manipulação de eventos.

Previsão do Tempo: Consumo de API REST (OpenWeatherMap) e tratamento de dados JSON em tempo real.

Site Android: Domínio de CSS avançado e estruturação semântica.
